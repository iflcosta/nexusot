# nexusot
Um NotebookLM voltado para a gestão de um OTServ 8.6


O NotebookLM foi desenvolvido a partir de fontes relacionadas ao desenvolvimento e gestão de um servidor privado de Tibia (OTServ).
A versão utilizada na pesquisa é o client 8.60, baseado no pacote TFS 1.5, e um datapack Baiak. Uma base moderna para jogadores que preferem a fluidez da versão 8.6.
